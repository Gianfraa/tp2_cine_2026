### Paginas - Grupo N°8

### Alejo Sanger — Index
Desarrollé la página principal del sitio. Mi objetivo fue ofrecer una vista clara de la cartelera, mostrando cada película con su póster, nombre, género y dos acciones disponibles: ver el detalle o comprar entradas.

### Román Strizzi — Detalle
Para la página de detalle busqué mostrar toda la información relevante de una película en un solo lugar: póster, título, puntuación con estrellas, sinopsis, duración, género, director, clasificación y horarios disponibles. También incluí el botón de compra como acción principal de la página.

### Nazareno Negrete — Estrenos
Mi idea fue presentar los próximos lanzamientos de forma prolija y simple, sin complejidad innecesaria. Armé una grilla de cards verticales donde cada una muestra el póster de la película y su fecha de estreno estimada.

### Julián Peralta — Login
Diseñé el formulario de acceso para clientes registrados. Incluye los campos de email y contraseña, redirige al perfil al ingresar correctamente y permite navegar al registro en caso de no tener cuenta.

### Gianfranco Tarulli — Registro
Para el formulario de registro busqué mantener una estética coherente con el resto del sitio. Incluí los datos básicos del usuario y una sección adicional con inputs especiales: selector de color favorito para las butacas, un range para la cantidad habitual de entradas, un select para el género cinematográfico preferido y radio buttons para elegir el snack favorito.

### Nicolás Castellini — Perfil
Desarrollé el panel de usuario, donde se muestran los datos del cliente logueado: foto de perfil, nombre, email, fecha de nacimiento y año de membresía. También agregué una sección con el historial de películas ya vistas y la fecha en que cada una fue vista.
